# An Indonesian News Sentiment Index for CPI Nowcasting (INSEI)

This project establishes a reproducible framework for native-language sentiment analysis in emerging market inflation nowcasting, aiming to overcome the latency of official CPI reports, the methodological flaw of relying on English translation, & improving from the monolithic concept of inlfation. The outcome is the **Indonesia News Sentiment of Inflation (INSEI)** index, validated as a leading indicator of Indonesia's monthly CPI rate.

## How to read the folder Code:

This project consists of multiple Jupyter Notebooks organized across several directories. To ensure the data pipeline and analysis remain consistent, please follow the execution order detailed below.
